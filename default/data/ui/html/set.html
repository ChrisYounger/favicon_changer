If you have a search-head cluster, its a good idea to set the same icon for all the servers - This can be done either by navigating to each search head manually or by running the below command from the monitoring console and setting the splunk_server= argument (The monitoring console must be setup correctly to talk to all search heads.).

| rest splunk_server=<NAME_OF_SEARCH_HEAD> /services/favicon_changer icon=<ICON_COLOR_NAME>
